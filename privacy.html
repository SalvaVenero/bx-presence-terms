Politique de confidentialité du bot Discord Bx 'Présence

Dernière mise à jour : 23/06/2025
Contact : salvavenero (Discord)

1. Données collectées

Le bot Bx 'Présence ne collecte pas de données personnelles telles que les noms, adresses email, adresses IP ou messages privés.

Les seules données utilisées sont :

  • Les identifiants (ID) des utilisateurs, serveurs, rôles et messages
  • Les réponses aux présences (✅, ❌, ⌛) avec la date d'interaction
  • La configuration des salons et rôles définis par les administrateurs

2. Utilisation des données

Les données sont utilisées uniquement pour :

  • Afficher les messages de présence
  • Créer des historiques hebdomadaires visibles par les membres du serveur
  • Identifier les membres n'ayant pas réagi pour les pings automatiques

3. Stockage des données

Les données sont traitées uniquement en mémoire pendant que le bot est en ligne. Aucune base de données externe ni sauvegarde permanente n’est utilisée.

4. Partage des données

Aucune donnée n’est partagée, vendue ou transmise à des tiers.

5. Droits des utilisateurs

Les utilisateurs peuvent demander la suppression de leurs données en contactant le développeur à tout moment via Discord.

6. Contact

Pour toute question ou demande relative à la vie privée, contactez : salvavenero (Discord)

Merci de faire confiance au bot Bx 'Présence.
